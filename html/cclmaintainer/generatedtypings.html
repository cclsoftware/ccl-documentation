Generated Typings

Classmodels can be converted to TypeScript typings (.d.ts files) using a CCL classmodel converter script tool. The general approach is to provide a set of types as .classmodel file and then use it as input for the TypeScript converter. See the script tools section for details on how to setup and use the CCL TypeScript converter script tool.
