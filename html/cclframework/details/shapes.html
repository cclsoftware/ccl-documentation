Geometric Shape Definitions

Note

How the rect width is determined:

template <typename Type>
inline Type TRect<Type>::getWidth () const
{ return right - left; }

Shape 1

Code

Rect r (1, 1, 9, 9);
// r.getWidth () == 8
IGraphics::fillRect (r, brush)
IGraphicsPath::addRect (r)
QuartzPath::fill (device, brush)

// same area for:
// IGraphics::addClip (r)
