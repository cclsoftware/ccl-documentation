virtual void CCL_API registerFilter (IConvertFilter* filter)

Register conversion filter.

The filter will be shared!

virtual void CCL_API unregisterFilter (IConvertFilter* filter)

Unregister conversion filter.

virtual void CCL_API registerImporter (IImportFilter* filter)

Register import filter.

The filter will be shared!

virtual void CCL_API unregisterImporter (IImportFilter* filter)

Unregister import filter.

virtual void CCL_API registerExporter (IExportFilter* filter)

Register export filter.

The filter will be shared!

virtual void CCL_API unregisterExporter (IExportFilter* filter)

Unregister exporter filter.

virtual tbool CCL_API canConvert (IUnknown* object, UIDRef cid = kNullUID) const

Check if object can be converted to given type.

virtual IUnknown*CCL_API convert (IUnknown* object, UIDRef cid = kNullUID) const

Convert object to given type.

In case of success, the new object is owned by the caller.

    UIDRef cid = kNullUID

Check if object of given type can be created from data stream.
