enum CCL::Document::Flags

Overview

#include <document.h>

enum Flags
{
    kImported            = 1 < <0,
    kOlderFormat         = 1 < <1,
    kSilent              = 1 < <2,
    kCanceled            = 1 < <3,
    kIsAutoSave          = 1 < <4,
    kIsSaveToNewFolder   = 1 < <5,
    kIsExportToNewFolder = 1 < <6,
    kSafeMode            = 1 < <7,
    kIsExport            = 1 < <8,
    kIsLoadingTemplate   = 1 < <9,
    kIgnoreDirtyUI       = 1 < <10,
    kSavingSuspended     = 1 < <11,
    kIsTemporary         = 1 < <12,
};

Detailed Documentation

Enum Values

kImported

document has been imported

kOlderFormat

document has been loaded from an older format version

kSilent

suppress any warning dialogs

kCanceled

document load has been canceled

kIsAutoSave

auto-save in progress

kIsSaveToNewFolder

save to new folder in progress

kIsExportToNewFolder

export to new folder in progress

kSafeMode

load with safety options

kIsExport

export in progress

kIsLoadingTemplate

loading from a document template in progress,

kIgnoreDirtyUI

do not mark document dirty if the data model is not modified

kSavingSuspended

document is currently not allowed to be saved

kIsTemporary

document is temporary: it will be deleted from disk when closed; flag is automatically reset on save
