enum CCL::RunLevel

Overview

Levels of initialization/termination phase. More…

#include <initterm.h>

enum RunLevel
{
    kFrameworkLevelFirst  = 0,
    kFrameworkLevelSecond = 10,
    kFrameworkLevelLast   = 20,
    kFirstRun             = 100,
    kSecondRun            = 1000,
    kLastRun              = 10000,
    kSetupLevel           = kSecondRun - 200,
    kAppLevel             = kSecondRun - 100,
};

Detailed Documentation

Levels of initialization/termination phase.

Enum Values

kFrameworkLevelFirst

(reserved for framework initialization)

kFirstRun

e.g. for loose binding of objects

kSecondRun

default level

kLastRun

after everything important is done

kSetupLevel

before application components are initialized

kAppLevel

application component level
