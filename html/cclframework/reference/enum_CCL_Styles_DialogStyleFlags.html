enum CCL::Styles::DialogStyleFlags

Overview

#include <controlstyles.h>

enum DialogStyleFlags
{
    kDialogButtonBase = 1 < < (kImageViewLastFlag + 1),
    kCancelButton     = kDialogButtonBase < <DialogResult::kCancel,
    kOkayButton       = kDialogButtonBase < <DialogResult::kOkay,
    kCloseButton      = kDialogButtonBase < <DialogResult::kClose,
    kApplyButton      = kDialogButtonBase < <DialogResult::kApply,
    kDialogOkCancel   = kOkayButton|kCancelButton,
};

Detailed Documentation

Enum Values

kCancelButton

user input is discarded, dialog is closed

kOkayButton

user input is confirmed, dialog is closed

kCloseButton

user input is discarded, dialog is closed

kApplyButton

user input is confirmed, dialog stays open
