enum CCL::IDocument::EventCode

Overview

Event codes for IDocumentEventHandler. More…

#include <idocument.h>

enum EventCode
{
    kCreated          = 0,
    kBeforeLoad,
    kLoadFinished,
    kLoadFailed,
    kBeforeSave,
    kSaveFinished,
    kClose,
    kActivate,
    kDeactivate,
    kViewActivated,
    kDestroyed,
    kBeforeAutoSave,
    kAutoSaveFinished,
};

Detailed Documentation

Event codes for IDocumentEventHandler.

ATTENTION: Keep constants in cclapp.js in sync!

Enum Values

kCreated

new document

kBeforeLoad

before loading document

kLoadFinished

after load finished

kLoadFailed

document load failed

kBeforeSave

before save

kSaveFinished

after save finished

kClose

on document close

kActivate

on document activate

kDeactivate

on document deactivate

kViewActivated

after document view is fully visible

kDestroyed

document is going away

kBeforeAutoSave

before auto save of document (also used for: save new version, save as template)

kAutoSaveFinished

after auto save of document
