Detailed Documentation

Root node in a 3D scene.

Methods

virtual void CCL_API setHandler (ISceneHandler3D* handler) = 0

Set scene handler (optional, not shared).

virtual ISceneHandler3D*CCL_API getHandler () const = 0

Get scene handler.

virtual void CCL_API beginEditing () = 0

Start editing a scene.

Changing visible properties of scene nodes or adding/removing nodes to/from a scene is only allowed within a beginEditing/endEditing block.

virtual void CCL_API endEditing (
    ISceneNode3D* node = nullptr,
    int editFlags = 0
) = 0

End editing a scene, signals a kChanged message.

Parameters:

node

A node that has beeen edited. If nullptr, any node may have been edited.

editFlags

Edit flags.
